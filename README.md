# SEO Audit – TD Consulting

## Overview

This repository contains a comprehensive SEO audit of the TD Consulting website.

**Website Audited**

https://www.td-consulting.ca/

## Objective

The goal of this project is to evaluate the website's search engine optimization performance and recommend improvements that increase visibility, traffic, and user experience.

---

## Scope of Audit

- Technical SEO
- On-Page SEO
- Local SEO
- Mobile Optimization
- Website Performance
- User Experience
- Content Quality

## Tools Used

- Google Lighthouse
- GTmetrix
- Google Search
- Manual SEO Inspection
